### File-Lib
  * Helpers to interact with the host machines file system

### Dependencies
* [app-root-path](https://github.com/inxilpro/node-app-root-path)
* [@keg-hub/jsutils](https://github.com/KegHub/keg-hub/tree/develop/repos/jsutils)

## Install

  * Install with 'yarn' or 'npm'
    ```js
      // With Yarn
      yarn add @keg-hub/file-lib
      // With NPM
      npm install @keg-hub/file-lib
    ```
  * Add to your code
    ```js

      // * Require code
      const {  } = require('@keg-hub/file-lib')



    ```
